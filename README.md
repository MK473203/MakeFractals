# MakeFractals

An application for exploring and generating images of fractals

## Hotkeys

|Hotkey|Function|
|---|---|
|W A S D|Moving around|
|Shift (hold)|Slower movement|
|* and /|Zooming in/out|
|+ and -|Changing iteration limit|
